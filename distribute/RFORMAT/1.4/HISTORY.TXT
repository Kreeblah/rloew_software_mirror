Version 1.4    Released 11/04/10

                  Revised Warning Messages.
                  Added TBPLUS Documentation.

Version 1.3    Released 10/19/10

                  Corrected alignment code for Extended MBR Partitions.
                  Added Warnings for improper lengths of Extended MBR Partitions.
                  Placed Warning Messages after Partition Parameter Display.
                  Added ExtMBR Format Start Address to Display.
                  Limited Default Cluster Size to 32KB.
                  Added Documentation for USB Drivers and Bad Sectors.

Version 1.2    Released 09/20/10

                  Corrected Sector Count in FAT32 Large Sector Boot Code
                  Added MBR Bypass Offset Code in 40-Bit Boot Sector Patch
                  Added 40-Bit Mapping when writing to Physical Drives
                  Added Primary and Logical Sector #'s to Physical Drive Mode

Version 1.1    Released 07/11/10

                  Corrected Documentation for -DOS6 Option
                  Set lower Default Cluster Size for small Partitions
                  Redesigned Alignment Option to minimize use of Offset mod,
                  Revised and extended -BootOnly Option.
                  Set Default Geometry to Values Read by INT 13H.
                  Modified Boot Sectors to support Large Logical Sectors.
                  Modified Boot Sectors to correct Error String Pointers.
                  Modified Boot Sectors to support IO.SYS Loader to Boot
                   from Large Logical Sectors.
                  Modified Boot Sector Writer to support Large Logical Sectors.
                  Updated Prompts.
                  Added Offset to Physical Mode.
                  Removed "New" from Display when No Parameters are changed. 
                  Added Boot Drive (Unit #) Option.

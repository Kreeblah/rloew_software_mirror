The R. Loew Electronics Disk Partitioning Utility provides the ability to
precisely partition a Hard Drive using one or more Boot profiles.



FEATURES

Supports Hard Drives up to 2100GB. Larger with additional Software. *
Repartition without erasing existing data.
Supports Multiple Boot Profiles.
Supports Multiple Primary Partitions on a Drive.
Supports Non-DOS Partitions.
Supports USB and FireWire Drives.
Multiple Boot supported without use of an Overlay.
Default Reboot to last used Profile.
Partition can be set to any size. Operating System limits useable sizes.
Optional support available for FAT16 Partitions up to 8GB or larger. *
Overlay Manager to support Add-ons.
Support for Large Sector Drives up to 32KB per Sector. *
Extended MBR Support for Hard Drives up to 32768TiB. *

* Additional Software support required to use these features.


REQUIREMENTS

DOS, Windows 95, 98, 98SE or ME.
BIOS or DDO Support for the size of Hard Drive required to run in normal mode.
TeraByte Plus (TBPLUS) Package if using Extended Features.

DEMO VERSION

The Demo Version supports all of the Commands except the Write, PutMBR and
Xcize Commands. This allows you to try out the Program but you will not be
able to change any of the Partitions on the Hard Drive. You will be able to
Install and Uninstall the Multi Boot MBR but only Profile "0" will be
available.

The RAM Limitation Patch Program Upgrade Package updates a previous RAM 
Limitation Patch Package. The Upgrade Program transfers the License Serial
Number from the Old Version of the Patch Program to the new one and replaces
the old Patch Program.



REQUIREMENTS

Earlier Version of the Full Version of the RAM Limitation Patch.



CONTENTS

PATCHMOD.EXE    Patch Update Program
PATCHMOD.DAT    Patch Update Data File for new Version
READ1ST.TXT     This File

Full RAM Limitation Patch Package except PATCHMEM.EXE File.



LICENSE

Read the Licensing terms and conditions in the LICENSE.TXT file before
using the Patch Upgrade Program. Installation of the Patch Upgrade
constitutes acceptance of all of the terms and conditions in the updated
LICENSE.TXT file.



INSTALLATION

1. If you have not setup an earlier Version of the RAM Limitation Patch,
   Unzip the Earlier Package and place the files in a folder.
2. If you have Installed the earlier Version of the RAM Limitation Patch,
   remove the Patch by running the PATCHMEM Program.
3. Unzip and copy the files from this Package OVER the previous Package.
4. Run the PATCHMOD.EXE Program to Upgrade the Patch Program.
5. Run the PATCHMEM.EXE Program to Install the upgraded RAM Limitation Patch.
   Read the upgraded MANUAL.TXT File before Installing the Upgrade.

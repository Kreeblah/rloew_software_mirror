This Patched Port Driver File replaces the Port Driver File installed by the
VIA MiniPort Driver Package Version 3.20B. This Driver File corrects errors
in the VIA Driver that affect Hard Drives larger than 137GB.


REQUIREMENTS

Windows 98, 98SE, or ME.
VIA MiniPort Driver Version 3.20B installed,
High Capacity Disk Patch Package or separate License.

BIOS support for hard drives >137GB or
A Windows Compatable Boot Manager that supports Hard Drives >137GB.
A Mini Boot Manager is now available from Rudolph R. Loew

NOTE: Some documentation and utilities may report this limit as 128GB.



CONTENTS

VIADSK.MPD      Patched VIA MiniPort Driver Version 1.0
README.TXT      This File



INSTALLATION

1. Copy the VIADSK.MPD file over the existing VIADSK.MPD in the
   SYSTEM\IOSUBSYS Folder in your Windows Directory.
2. Reboot your Computer for the update to take effect.

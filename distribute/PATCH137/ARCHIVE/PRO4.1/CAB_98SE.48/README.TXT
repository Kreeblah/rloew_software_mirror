This Prepatched Cabinet File replaces the corresponding Cabinet file on
the Windows 98SE Installation CD. Installing Windows from the modified CD
or Image will automatically Install the Patched Port Driver required to
provide direct support for Hard Drives larger than 137GB without requiring a
controller card or the Intel Application Accelerator which can only be used
with some Intel Chipset Motherboards.

This Cabinet file is intended to be used with the High Capacity Disk Patch
Package.



REQUIREMENTS

Windows 98SE Installation CD.
High Capacity Disk Patch Package.

BIOS support for hard drives >137GB or
A Windows Compatable Boot Manager that supports Hard Drives >137GB.
A Mini Boot Manager is now available from Rudolph R. Loew

NOTE: Some documentation and utilities may report this limit as 128GB.

INSTALLATION

1. Copy the files for your WIndows 98SE Installation CD to a folder on your
   Hard Drive.
2. Copy the enclosed Cabinet File over the original file in the WIN98 folder
   within the folder you copied the CD into.
3. You can burn a new CD from the folder you copied the CD into, or you can
   Install Windows using the files on the Hard Drive.
4. Install Windows 98 SE from the CD created in step 3 or using the files in
   the Hard Drive.
5. If you want to be able to Uninstall or Upgrade the Patch after
   installation, perform the following steps:
6. Extract the ESDI_506.BAK File in this Cabinet File. The following command
   will Extract the File:

   EXTRACT WIN98_51.CAB ESDI_506.BAK

7. Copy the ESDI_506.BAK File into the SYSTEM\IOSUBSYS folder in the Windows
   Directory

The High Capacity Disk Patch Program Upgrade Package updates a previously
installed High Capacity Disk Patch Package. The Upgrade Program transfers
the License Serial Number from the Old Version of the Patch Program to the
new one and replaces the old Patch Program.



REQUIREMENTS

Earlier Version of the Full Version of the High Capacity Disk Patch.



CONTENTS

PATCHMOD.EXE    Patch Update Program
PATCHMOD.DAT    Patch Update Data File for new Version
READ1ST.TXT     This File

Full High Capacity Disk Patch Package except PATCHATA.EXE File.

LICENSE.TXT file.



INSTALLATION

1. If you have not Installed an earlier Version of the High Capacity Disk
   Patch, Unzip the Earlier Package and place the files in a folder.
2. If you have Installed an earlier Version of the High Capacity Disk
   Patch, remove the Patch by running the PATCHATA Program.
3. Unzip and copy the files from this Package OVER the previously installed
   Package.
4. Run the PATCHMOD.EXE Program to Upgrade the Patch Program.
5. Run the PATCHATA.EXE Program to Install/Upgrade the ESDI_506.PDR Port
   Driver. Windows ME users must perform this step from DOS or from Windows
   Safe Mode. See the MANUAL.TXT File for more information.
6. Update any other copies of the PATCHATA.EXE (such as for Automatic
   Installation/Verification) from the Upgraded PATCHATA.EXE Program.
